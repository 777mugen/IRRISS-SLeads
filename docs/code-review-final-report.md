**修复模板**:
1. ✅ `query/search.html` - DOI 查询结果（最危险）
2. ✅ `batch/monitor.html` - 批处理监控（最危险）

**提交**: `cc0494a`
**影响**: 消除最危险的 XSS 攻击向量（50% 模板）

### ✅ #003 - 修复 N+1 查询 (20 分钟, 2/6 处)
