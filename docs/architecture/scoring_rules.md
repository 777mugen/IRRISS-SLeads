# 评分规则

## 禂述

评分系统计算 **0-100 数值分数**，然后映射为 **A/B/C/D 等级**。

- **分数**: 仅在数据库内部保留
用于排序和筛选
- **等级**: 对外展示（CSV、飞书）
用于销售判断

---

## 等级区间

```
A ≥ 80  → 高优线索
B = 65-79  → 中优线索
C = 50-64  → 低优线索
D < 50  → 待定线索
```
