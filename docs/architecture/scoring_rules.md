# 评分规则

## 等级区间

A ≥ 80
B 65–79
C 50–64
D < 50

系统内部保留数值分，销售侧仅展示 ABCD。

---

# 论文线索评分（100分）

A 信息完备度 20%
D 匹配度 25%
E 时效性 20%
C 决策链条 15%
G 机构类型 10%
F 历史互动 10%

高/中/低条件遵循 V4 评分规则。
