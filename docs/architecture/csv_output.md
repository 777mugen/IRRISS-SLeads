# CSV 输出规格

## 输出周期

| 周期 | 类型 | 规则 |
|------|------|------|
| 每日 | 增量 | 仅新增和变更的线索 |
| 每周日 | 全量 | 先与数据库同步，diff 字段标注变更标记 |

## 全量导出 diff 标注

每周日全量导出时，每条记录增加 `变更标记` 字段：
- 新增：本周新入库的线索
- 已更新：本周有字段变更的线索
- 无变化：与上次全量导出一致

目的：防止销售在全量表中无法分辨哪些是新变化。
