5. 提升速度

### 参数配置优化

**新增 Headers**:
```python
headers = {
    'Authorization': 'Bearer {api_key}',
    'Accept': 'text/plain',
    'X-Respond-With': 'markdown',
    'X-Respond-Timing': 'resource-idle',      # 平衡速度和完整性
    'X-Timeout': '60',
    'X-Engine': 'browser',                    # 模拟浏览器，减少反爬
    'X-Cache-Tolerance': '3600',              # 1小时缓存
    'X-Remove-Selector': 'nav, aside, footer, .sidebar, .advertisement, .comments, .related-articles, .social-share, img, a img, figure',
    'X-Retain-Links': 'none',                 # ✅ 去除链接
    'X-Retain-Images': 'none',                # ✅ 去除图片
    'X-With-Generated-Alt': 'false',
    'X-Locale': 'en-US',
    'X-Referer': 'https://doi.org/',          # 模拟从 DOI 跳转
    'X-Token-Budget': '50000',
    'X-Robots-Txt': 'false'
}
```

### 关键改进
- ✅ `X-Engine: browser` - 模拟浏览器，减少反爬（虽然慢一点）
- ✅ `X-Retain-Links: none` - 去除链接（用户要求）
- ✅ `X-Retain-Images: none` - 去除图片（用户要求）
- ✅ `X-Respond-Timing: resource-idle` - 平衡速度（比 network-idle 快）
- ✅ `X-Cache-Tolerance: 3600` - 利用缓存提升速度
