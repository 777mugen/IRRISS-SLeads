## 📝 技术决策

### 技术栈选择
- **后端**: FastAPI (异步性能)
- **模板**: Jinja2 (服务器渲染)
- **前端交互**: HTMX (声明式AJAX)
- **样式**: Tailwind CSS (实用工具类)
- **图表**: ECharts (支持中国地图)

### 架构设计
- **单页多标签**: 更好的用户体验（内部工具）
- **三层架构**: 路由层 → API层 → 服务层
- **实时刷新**: HTMX轮询（60秒），未来可升级WebSocket
- **CSV流程**: 上传 → 预览 → 确认 → 更新

### 安全设计
- **无认证**: 内部工具，Basic Auth可选
- **输入验证**: Pydantic模型验证
- **SQL注入防护**: SQLAlchemy ORM
- **XSS防护**: Jinja2自动转义
